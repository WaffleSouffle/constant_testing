# Constant Testing
## Monitors elixir projects and runs tests when files are edited

### Usage
1. Setup for your Elixir project:
  ```shell
 ./setup_in.sh ~/src/my_lovely_project
 ```

2. Run in that projects folder
  ```shell
  cd ~/src/my_lovely_project
  ./constant-testing.sh
  ```

3. Start writing code!
